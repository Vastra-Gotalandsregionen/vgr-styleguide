Print _print.scss

The styleguide includes print styles developed by HTML5 Boilerplate to give you some basic print-specific styles. The print CSS are activated trough a media query. It includes:

  • Clearing out backgrounds, box shadows and text shadows
  • Appending link URLs after the anchor text
  • Adding borders to <blockquote> and <pre> elements
  • Page cleanup and window minimization
