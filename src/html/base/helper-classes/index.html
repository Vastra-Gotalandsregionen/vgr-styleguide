Helper Classes _helpers.scss

We include some helpful classes to add common positioning behaviors to elements.

Aligment Classes

Class Description
.vgr-align-left Floats the element left.
.vgr-align-right Floats the element right.
.vgr-align-center Aligns the element center.

Visibility Classes

Can also be used as a mixin.

Class Description
.vgr-visually-hidden Hide for screen users, read for screen reader users
.vgr-visually-hidden--focusable Show only for screen readers

Clearfix

Clear floats by adding .vgr-clearfix to the parent element of the floated elements. Can also be used as a mixin.

Class Description
.vgr-clearfix Clear parent of floated children.
