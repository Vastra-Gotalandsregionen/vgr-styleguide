Use the color palette. If you need custom colors do not forget that they have to be adapted for the web and contrast from the accessibility aspect.

The color contrast for text and interactive elements must follow WCAG 2.0 AA.

WCAG ensure that content is accessible by everyone, regardless of disability or user device. To meet these standards, text and interactive elements must have a color contrast ratio of at least 4.5:1. This ensures that viewers who cannot see the full color spectrum are able to read the text.

If you choose to customize beyond this palette, this color contrast tool is a useful resource for testing the compliance of any color combination.
