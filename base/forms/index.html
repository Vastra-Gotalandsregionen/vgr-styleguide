Forms _forms.scss

Keep forms as simple as possible – only ask what's needed to run your service.

Use the .form-label class to get default margins, layout and styling for form labels. Additionally, use the .vgr-form__label--required class around an asterisk to declare a field required. If a field needs to be marked as optional, wrap "(optional)" in .vgr-form__label--optional.
