Grid _layout-classes.scss

The Grid is based on a 12 column layout where each column, including the gutter is 100 pixels in on a large screen. The grid is fluid and each column has a default gutter of 15 pixels on each side. The default totalt maximum width of the grid is 1230 pixels.

Variable settings

Variable Default value
$max-width 1440px
$gutter-width 15px
