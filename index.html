Web Standards & Styleguide for Västra Götalands­regionen

Guidelines and collections of open source UI components and visual identify to create consistency and beautiful user experiences across Västra Götalands­regionen's websites.

Accessibility

Accessibility is a first class concern and as such each component within the framework has been carefully crafted to be fully accessible to keyboard and screenreaders.

För beställare

Instruktioner för dig som beställare, eller projektledare, där ni kommer ta fram eller påverka en webbplats.

For designers

Guidelines regarding designers contributions to the style-guide and the pattern library.

For developers

The style-guide consists both of guidelines and code samples for you as a frontend.

Prestandabudget

Krav på gränssnittskod, tillgänglighet och hur snabbt en webbplats ska ladda in hos en användare.

Compatibility

VGR Base was designed for and tested on the most used browsers and devices. See the rundown on what's tested, and what to do if something misbehaves.
